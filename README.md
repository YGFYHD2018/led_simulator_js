# led_simulator_js (3D LED CUBE Simulator)

## Requiments

Ruby 2.5.x  
Web Browser  

## How to Setup

1. `$ git clone https://github.com/YGFYHD2018/led_simulator_js.git`  
2. `$ cd led_simulator_js`  
3. `$ cd bundle install`  

## How to Run

1. `$ bundle exec ruby app.rb`
2. Open URL "http://localhost:4567" with a browser.
